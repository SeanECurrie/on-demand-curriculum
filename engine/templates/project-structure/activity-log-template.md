# Activity Log

Lightweight breadcrumbs. What happened, when.

**Decision tree:** Log an entry when you complete a discrete action — research query executed, file written, report completed, phase finished, skill invoked, decision made. If you're unsure whether something is an activity-log entry or an intelligence-log entry, ask: "Is this a strategic insight or a record of what I did?" Strategic insights go to `intelligence-log.md`. Everything else goes here.
